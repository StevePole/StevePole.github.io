What is a BIC / Swift code?

A
Bank code A-Z

Consisting of 4 character, letter only

B
Country code A-Z

ISO 3166-1 alpha-2 of letters only

C
Location code 0-9 A-Z

Will have "1" in the second character

D
Branch Code 0-9 A-Z

Optional 'XXX' for primary office

AAAA BB CC DD
BIC / SWIFT CODE

Swift Code is a standard format of Bank Identifier Codes (BIC) and it is unique identification code for a particular bank. These codes are used when transferring money between banks, particularly for international wire transfers.

Banks also used the codes for exchanging other messages between them.
